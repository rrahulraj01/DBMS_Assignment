# 📊 DB Lab 4 — Data Retrieval Using SQL

## 📌 Objective

This lab focuses on practicing **read-only SQL queries** to retrieve and analyze data from existing database tables.
It covers filtering, sorting, aliasing, and generating derived outputs without modifying the database.

## 🧠 Topics Covered

* Data retrieval using `SELECT`
* Column aliases (`AS`)
* Filtering with `WHERE`
* Sorting with `ORDER BY`
* Limiting results using `LIMIT`
* Derived/computed columns
* Working with date functions

## 🗂️ Lab Structure

### Part A — Basic Data Display

Use column aliases and display selected fields from tables like **Student, Faculty, Course, Enrollment**.

### Part B — Conditional Data Display

Retrieve records based on conditions such as:

* Department
* Gender
* Designation
* Credits
* Date of Birth

### Part C — Sorting & Limiting

Sort records in ascending/descending order and limit the number of rows returned.

### Part D — Derived Output

Generate computed values such as:

* Student age from DOB
* Updated course credits
* Extracted year and email domain

## 🛠️ Tools Used

* SQL (ORACLE SQL)

## ✅ Outcome

By completing this lab, you will be able to write efficient **read-only SQL queries** to analyze and present database information clearly.
